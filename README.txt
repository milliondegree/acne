使用说明：
1、打开浏览器，在地址栏输入 https://60.205.182.221 浏览器会提示该网站的安全证书不安全，选择 忽略危险继续访问 进入登录界面。（chrome浏览器需要先点击 高级 才能出现继续访问的选项）
2、在登录界面输入用户名000000、密码000000，点击登录，进入图像采集界面。
3、第一次登录浏览器会询问是否允许调用摄像头，点击“是”或者“允许”，界面的左侧会出现摄像头图像。
4、将摄像头对准人脸问题部位（额头、脸颊等），点击屏幕中间的“拍摄”按钮，会在摄像头图像右侧出现刚刚拍摄的图片，右键该图片，选择另存为，名称为六类痤疮的首字母小写缩写（囊肿：nz；脓包：nb；黑头：ht；白头：bt；结节：jj；丘疹：qz）+任意其他符号，文件格式为jpg。例如：nz123.jpg, qz.jpg, ht000.jpg 
5、点击“选择图片”，出现文件选择对话框，选择之前保存的图片，点击“上传”，界面刷新表明上传成功，上传不符合命名格式的图片虽然不会报错，但是会被忽略，可以一次性选择多个图片上传，但是不鼓励一次性超过10张，这可能会引发难以预知的风险。
6、改系统还处于开发状态，可能存在诸多风险或不稳定性，请使用者多多包涵，积极反馈，谢谢！
